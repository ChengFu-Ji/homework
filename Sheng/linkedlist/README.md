# Linked list #

Linked list 是一種資料結構，每一個資料結構的型態稱作 Node，
Linked list 的基礎結構就是 data + 指向下一個 Node 的指標 。
在實作 linked list 時，可以依據需求實作出將 Node 加在前面或是加在後面的 Linked list。

### 目前功能 ###
1. add 功能
2. del 功能
3. load 功能
4. save 功能
5. showlist 功能
6. commandinput 功能
7. showN 功能
8. cleanlist 功能
